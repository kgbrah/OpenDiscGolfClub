# PDGA Member Links

PDGA links are optional. Add a `pdga` number to a member and enable links:

```json
{
  "members": {
    "pdgaLinks": true,
    "items": [
      {
        "firstName": "Alex",
        "lastName": "Rivera",
        "yearJoined": 2024,
        "pdga": "123456"
      }
    ]
  }
}
```

Members without a PDGA number render normally without a link.
